# MCP Proxy v1 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Ship `AI.Sentinel.Mcp` library + `sentinel-mcp proxy --target <cmd> [<args>...]` CLI that wraps a target MCP server subprocess, intercepts `tools/call` and `prompts/get` via SDK-native `McpRequestFilter`s, and turns threats into JSON-RPC errors.

**Architecture:** One process. `McpServer` on our stdin/stdout (host-facing), `McpClient` connected via `StdioClientTransport` to the target subprocess. Two `McpRequestFilter<TReq, TRes>` instances registered on the server run Sentinel before+after each intercepted call; default handlers forward everything else to the upstream client. Security-only detector preset; env-var config reused from hooks. Design reference: [docs/plans/2026-04-24-mcp-proxy-design.md](docs/plans/2026-04-24-mcp-proxy-design.md).

**Tech Stack:** `ModelContextProtocol 1.2.*` (Core) — typed MCP session, transports, filters. `AI.Sentinel` core (`SentinelPipeline.ScanMessagesAsync`, `HookConfig`). `AI.Sentinel.ClaudeCode` (reuse `HookConfig`, `UnusedChatClient`). xUnit + existing `NonParallel` collection for env-var tests.

**Important context for the implementer:**

- The `ModelContextProtocol` 1.2.0 SDK API is rich. Some code below is illustrative; if a detail (exact method signature, property name) differs from what you find in the SDK, **adapt to the SDK, don't force the sketch**. The SDK is the source of truth.
- Every failing-test step uses `dotnet test --filter "FullyQualifiedName~TestName"` so you get fast feedback. Run the full suite only before committing.
- Commit after each task. Small, reviewable, reversible.
- Check compilation after each code change with a targeted `dotnet build` (only the affected project + deps, not the whole solution).
- The CLI will publish as **both** a `dotnet tool` NuGet package (default `dotnet pack`) and as a Native AOT binary (`dotnet publish -r <rid> -p:PublishAot=true`). The `PackAsTool` / `InvariantGlobalization` conditionals we use on the hook CLIs handle this cleanly.

**Step 3: Implement `MessageBuilder.cs`**

```csharp
using System.Text;
using System.Text.Json;
using Microsoft.Extensions.AI;
using ModelContextProtocol.Protocol;

namespace AI.Sentinel.Mcp;

/// <summary>Maps MCP protocol models to <see cref="ChatMessage"/> arrays for Sentinel scanning.</summary>
internal static class MessageBuilder
{
    private const string Separator = "\n---\n";
    private const string TruncationMarkerPrefix = " [truncated ";

    public static ChatMessage[] BuildToolCallRequest(CallToolRequestParams req)
    {
        var argsJson = SerializeArguments(req.Arguments);
        var text = $"tool:{req.Name} input:{argsJson}";
        return [new ChatMessage(ChatRole.User, text)];
    }

    public static ChatMessage[] BuildToolCallResponse(
        CallToolRequestParams req, CallToolResult result, int maxScanBytes)
    {
        var argsJson = SerializeArguments(req.Arguments);
        var requestText = $"tool:{req.Name} input:{argsJson}";
        var responseText = ExtractScannableText(result.Content, maxScanBytes);

        if (string.IsNullOrEmpty(responseText))
        {
            return [new ChatMessage(ChatRole.User, requestText)];
        }

        return
        [
            new ChatMessage(ChatRole.User,      requestText),
            new ChatMessage(ChatRole.Assistant, responseText),
        ];
    }

    public static ChatMessage[] BuildPromptGetResponse(
        GetPromptRequestParams req, GetPromptResult result, int maxScanBytes)
    {
        var flattened = FlattenPromptMessages(result.Messages, maxScanBytes);
        if (string.IsNullOrEmpty(flattened))
        {
            return [];
        }
        return [new ChatMessage(ChatRole.Assistant, flattened)];
    }

    private static string SerializeArguments(IDictionary<string, JsonElement>? args)
    {
        if (args is null || args.Count == 0) return "{}";
        return JsonSerializer.Serialize(args, McpJsonContext.Default.IDictionaryStringJsonElement);
    }

    private static string ExtractScannableText(
        IList<ContentBlock>? blocks, int maxScanBytes)
    {
        if (blocks is null || blocks.Count == 0) return string.Empty;

        var sb = new StringBuilder();
        var firstBlock = true;
        foreach (var block in blocks)
        {
            if (block is TextContentBlock text && !string.IsNullOrEmpty(text.Text))
            {
                if (!firstBlock) sb.Append(Separator);
                sb.Append(text.Text);
                firstBlock = false;
            }
        }

        return TruncateIfNeeded(sb.ToString(), maxScanBytes);
    }

    private static string FlattenPromptMessages(
        IList<PromptMessage>? messages, int maxScanBytes)
    {
        if (messages is null || messages.Count == 0) return string.Empty;

        var sb = new StringBuilder();
        var first = true;
        foreach (var msg in messages)
        {
            if (msg.Content is TextContentBlock text && !string.IsNullOrEmpty(text.Text))
            {
                if (!first) sb.Append(Separator);
                sb.Append(text.Text);
                first = false;
            }
        }

        return TruncateIfNeeded(sb.ToString(), maxScanBytes);
    }

    private static string TruncateIfNeeded(string text, int maxScanBytes)
    {
        if (text.Length <= maxScanBytes) return text;
        var truncated = text[..maxScanBytes];
        var removed = text.Length - maxScanBytes;
        return $"{truncated}{TruncationMarkerPrefix}{removed} chars]";
    }
}
```

> `ContentBlock`, `TextContentBlock`, `ImageContentBlock`, `PromptMessage.Content` — if the SDK exposes these differently (e.g. `Content` is polymorphic via a discriminator string rather than a class hierarchy), adapt the `is` pattern matches. The goal is: identify text-bearing blocks, concatenate their text.

**Step 2: Implement `FakeMcpServer`**

```csharp
using System.IO.Pipelines;
using System.Threading.Channels;
using ModelContextProtocol.Client;
using ModelContextProtocol.Protocol;
using ModelContextProtocol.Server;

namespace AI.Sentinel.Tests.Mcp;

/// <summary>
/// In-memory MCP server + paired client transports. Call <see cref="Start"/>
/// to boot; it returns an <see cref="IClientTransport"/> that the proxy under
/// test connects to. Canned responses are queued via <see cref="EnqueueToolResult"/>
/// / <see cref="EnqueuePromptResult"/>; if empty, defaults apply.
/// </summary>
public sealed class FakeMcpServer : IAsyncDisposable
{
    private readonly Channel<CallToolResult> _toolResults = Channel.CreateUnbounded<CallToolResult>();
    private readonly Channel<GetPromptResult> _promptResults = Channel.CreateUnbounded<GetPromptResult>();
    private McpServer? _server;
    private Task? _runTask;
    private CancellationTokenSource? _cts;
    private Pipe? _fromProxy;
    private Pipe? _toProxy;

    public List<CallToolRequestParams> ReceivedToolCalls { get; } = new();
    public List<GetPromptRequestParams> ReceivedPromptGets { get; } = new();

    public void EnqueueToolResult(CallToolResult result) => _toolResults.Writer.TryWrite(result);
    public void EnqueuePromptResult(GetPromptResult result) => _promptResults.Writer.TryWrite(result);

    /// <summary>Starts the fake server and returns a client transport the proxy can connect to.</summary>
    public IClientTransport Start()
    {
        _fromProxy = new Pipe();
        _toProxy   = new Pipe();

        var serverTransport = new StreamServerTransport(
            input:  _fromProxy.Reader.AsStream(),
            output: _toProxy.Writer.AsStream(),
            name:   "fake-mcp-server",
            loggerFactory: null);

        var options = new McpServerOptions
        {
            ServerInfo = new Implementation { Name = "fake-mcp-server", Version = "0.0.0" },
            Capabilities = new ServerCapabilities
            {
                Tools = new ToolsCapability(),
                Prompts = new PromptsCapability(),
            },
            Handlers = new McpServerHandlers
            {
                ListToolsHandler = (ctx, ct) => new ValueTask<ListToolsResult>(new ListToolsResult
                {
                    Tools =
                    [
                        new Tool { Name = "read_file",  Description = "Read a file." },
                        new Tool { Name = "write_file", Description = "Write a file." },
                    ],
                }),
                CallToolHandler = async (ctx, ct) =>
                {
                    ReceivedToolCalls.Add(ctx.Params!);
                    if (_toolResults.Reader.TryRead(out var queued)) return queued;
                    // Default: echo args as text
                    return new CallToolResult
                    {
                        Content = [new TextContentBlock { Text = $"called:{ctx.Params.Name}" }],
                    };
                },
                ListPromptsHandler = (ctx, ct) => new ValueTask<ListPromptsResult>(new ListPromptsResult
                {
                    Prompts = [new Prompt { Name = "onboard" }],
                }),
                GetPromptHandler = async (ctx, ct) =>
                {
                    ReceivedPromptGets.Add(ctx.Params!);
                    if (_promptResults.Reader.TryRead(out var queued)) return queued;
                    return new GetPromptResult
                    {
                        Messages =
                        [
                            new PromptMessage
                            {
                                Role = Role.Assistant,
                                Content = new TextContentBlock { Text = "welcome" },
                            },
                        ],
                    };
                },
            },
        };

        _server = McpServer.Create(serverTransport, options, loggerFactory: null, serviceProvider: null);
        _cts = new CancellationTokenSource();
        _runTask = _server.RunAsync(_cts.Token);

        // Return a client transport that reads from _toProxy (server's output) and
```

```csharp
using AI.Sentinel.ClaudeCode;
using Microsoft.Extensions.Logging.Abstractions;
using ModelContextProtocol.Client;
using ModelContextProtocol.Protocol;
using ModelContextProtocol.Server;

namespace AI.Sentinel.Mcp;

/// <summary>Runs the Sentinel MCP proxy — host ⇆ server side ⇆ filters ⇆ client side ⇆ target.</summary>
public static class McpProxy
{
    public static async Task RunAsync(
        ITransport hostTransport,
        IClientTransport targetTransport,
        HookConfig config,
        McpDetectorPreset preset,
        int maxScanBytes,
        CancellationToken ct)
    {
        // Connect to the upstream target
        await using var targetClient = await McpClient.CreateAsync(
            targetTransport,
            clientOptions: null,
            loggerFactory: NullLoggerFactory.Instance,
            cancellationToken: ct).ConfigureAwait(false);

        var pipeline = McpPipelineFactory.Create(config, preset);

        // Forward-only handlers (Task 7/8 will add filters on top)
        var handlers = new McpServerHandlers
        {
            CallToolHandler = async (req, c) =>
                await targetClient.CallToolAsync(
                    req.Params!.Name,
                    ToObjectDictionary(req.Params.Arguments),
                    cancellationToken: c).ConfigureAwait(false),

            GetPromptHandler = async (req, c) =>
                await targetClient.GetPromptAsync(
                    req.Params!.Name,
                    ToObjectDictionary(req.Params.Arguments),
                    cancellationToken: c).ConfigureAwait(false),

            ListToolsHandler = async (req, c) =>
            {
                var tools = await targetClient.ListToolsAsync(c).ConfigureAwait(false);
                return new ListToolsResult { Tools = tools.Select(t => t.ProtocolTool).ToList() };
            },

            ListPromptsHandler = async (req, c) =>
            {
                var prompts = await targetClient.ListPromptsAsync(c).ConfigureAwait(false);
                return new ListPromptsResult { Prompts = prompts.Select(p => p.ProtocolPrompt).ToList() };
            },

            ListResourcesHandler = async (req, c) =>
            {
                var resources = await targetClient.ListResourcesAsync(c).ConfigureAwait(false);
                return new ListResourcesResult { Resources = resources.Select(r => r.ProtocolResource).ToList() };
            },

            ReadResourceHandler = async (req, c) =>
                await targetClient.ReadResourceAsync(req.Params!.Uri, c).ConfigureAwait(false),
        };

        var serverOptions = new McpServerOptions
        {
            ServerInfo = new Implementation { Name = "sentinel-mcp", Version = "0.1.0" },
            Capabilities = new ServerCapabilities
            {
                Tools     = new ToolsCapability(),
                Prompts   = new PromptsCapability(),
                Resources = new ResourcesCapability(),
            },
            Handlers = handlers,
        };

        using var server = McpServer.Create(
            hostTransport,
            serverOptions,
            loggerFactory: NullLoggerFactory.Instance,
            serviceProvider: null);

        await server.RunAsync(ct).ConfigureAwait(false);
    }

    private static IReadOnlyDictionary<string, object?>? ToObjectDictionary(
        IDictionary<string, System.Text.Json.JsonElement>? args)
    {
        if (args is null) return null;
        var dict = new Dictionary<string, object?>(args.Count, StringComparer.Ordinal);
        foreach (var (k, v) in args) dict[k] = v;
        return dict;
    }
}
```

If the property names differ (`.ProtocolTool` → `.Protocol` or similar), adapt. The forward-pattern is: unwrap the client-side convenience, return the raw protocol model.

```csharp
using AI.Sentinel;
using AI.Sentinel.ClaudeCode;
using ModelContextProtocol;
using ModelContextProtocol.Protocol;
using ModelContextProtocol.Server;

namespace AI.Sentinel.Mcp;

internal static class ToolCallInterceptor
{
    public static McpRequestFilter<CallToolRequestParams, CallToolResult> Create(
        SentinelPipeline pipeline,
        int maxScanBytes,
        TextWriter stderr) =>
        next => async (ctx, ct) =>
        {
            var req = ctx.Params ?? throw new McpException("missing tool call parameters", McpErrorCode.InvalidParams);

            // Pre-scan (request direction)
            var requestMessages = MessageBuilder.BuildToolCallRequest(req);
            SentinelError? preError;
            try
            {
                preError = await pipeline.ScanMessagesAsync(requestMessages, ct: ct).ConfigureAwait(false);
            }
            catch (Exception ex)
            {
                // Fail-open — log to stderr, forward
                stderr.WriteLine($"[sentinel-mcp] event=tools/call decision=FailOpen phase=pre reason={ex.GetType().Name}:{ex.Message}");
                preError = null;
            }
            if (preError is SentinelError.ThreatDetected pre)
            {
                stderr.WriteLine($"[sentinel-mcp] event=tools/call decision=Block tool={req.Name} detector={pre.Result.DetectorId} severity={pre.Result.Severity} phase=request");
                throw new McpException($"Blocked by AI.Sentinel: {pre.Result.DetectorId} {pre.Result.Severity}: {pre.Result.Reason}", McpErrorCode.InternalError);
            }

            // Forward to the upstream target (via the next handler set in McpProxy)
            var result = await next(ctx, ct).ConfigureAwait(false);

            // Post-scan (response direction)
            var responseMessages = MessageBuilder.BuildToolCallResponse(req, result, maxScanBytes);
            SentinelError? postError;
            try
            {
                postError = await pipeline.ScanMessagesAsync(responseMessages, ct: ct).ConfigureAwait(false);
            }
            catch (Exception ex)
            {
                stderr.WriteLine($"[sentinel-mcp] event=tools/call decision=FailOpen phase=post reason={ex.GetType().Name}:{ex.Message}");
                postError = null;
            }
            if (postError is SentinelError.ThreatDetected post)
            {
                stderr.WriteLine($"[sentinel-mcp] event=tools/call decision=Block tool={req.Name} detector={post.Result.DetectorId} severity={post.Result.Severity} phase=response");
                throw new McpException($"Blocked by AI.Sentinel: {post.Result.DetectorId} {post.Result.Severity}: {post.Result.Reason}", McpErrorCode.InternalError);
            }

            stderr.WriteLine($"[sentinel-mcp] event=tools/call decision=Allow tool={req.Name}");
            return result;
        };
}
```

```csharp
using AI.Sentinel.ClaudeCode;
using AI.Sentinel.Mcp;
using ModelContextProtocol.Client;
using ModelContextProtocol.Server;

namespace AI.Sentinel.Mcp.Cli;

internal static class ProxyCommand
{
    /// <summary>Parses `proxy --target &lt;cmd&gt; [&lt;target-args&gt;...]` and runs the proxy.</summary>
    public static async Task<int> RunAsync(
        string[] args, TextReader stdin, TextWriter stdout, TextWriter stderr, CancellationToken ct)
    {
        // args[0] is "proxy" (already matched by Program)
        if (args.Length < 3 || args[1] != "--target")
        {
            await stderr.WriteLineAsync("Usage: sentinel-mcp proxy --target <command> [<target-args>...]").ConfigureAwait(false);
            return 1;
        }

        var targetCommand = args[2];
        var targetArgs = args.Skip(3).ToArray();

        var envVars = Environment.GetEnvironmentVariables()
            .Cast<System.Collections.DictionaryEntry>()
            .Where(e => e.Key is string k && (k.StartsWith("SENTINEL_HOOK_", StringComparison.Ordinal) || k.StartsWith("SENTINEL_MCP_", StringComparison.Ordinal)))
            .ToDictionary(e => (string)e.Key, e => e.Value as string, StringComparer.Ordinal);

        var config = HookConfig.FromEnvironment(envVars);
        var preset = ParsePreset(envVars);
        var maxScanBytes = ParseMaxScanBytes(envVars);

        // Target transport — spawn the subprocess via stdio
        var targetTransport = new StdioClientTransport(
            new StdioClientTransportOptions
            {
                Command = targetCommand,
                Arguments = targetArgs,
                Name = targetCommand,
            },
            loggerFactory: null);

        // Host transport — our stdin/stdout, speaking MCP to the MCP host that invoked us
        using var hostStdin  = Console.OpenStandardInput();
        using var hostStdout = Console.OpenStandardOutput();
        var hostTransport = new StdioServerTransport(
            serverName: "sentinel-mcp",
            loggerFactory: null);

        try
        {
            await McpProxy.RunAsync(
                hostTransport:   hostTransport,
                targetTransport: targetTransport,
                config:          config,
                preset:          preset,
                maxScanBytes:    maxScanBytes,
                stderr:          stderr,
                ct:              ct).ConfigureAwait(false);
            return 0;
        }
        catch (Exception ex) when (IsTargetFailure(ex))
        {
            await stderr.WriteLineAsync($"sentinel-mcp: target process failed: {ex.Message}").ConfigureAwait(false);
            return 2;
        }
    }

    private static McpDetectorPreset ParsePreset(IDictionary<string, string?> env)
    {
        if (!env.TryGetValue("SENTINEL_MCP_DETECTORS", out var value) || string.IsNullOrEmpty(value))
            return McpDetectorPreset.Security;
        return value.Equals("all", StringComparison.OrdinalIgnoreCase)
            ? McpDetectorPreset.All
            : McpDetectorPreset.Security;
    }

    private static int ParseMaxScanBytes(IDictionary<string, string?> env)
    {
        if (env.TryGetValue("SENTINEL_MCP_MAX_SCAN_BYTES", out var value)
            && int.TryParse(value, out var parsed) && parsed > 0)
            return parsed;
        return 262144; // 256 KB default
    }

    private static bool IsTargetFailure(Exception ex) =>
        ex is System.ComponentModel.Win32Exception
           or System.IO.FileNotFoundException
           or System.IO.IOException;
}
```

> `StdioServerTransport` may not have a `serverName` ctor parameter — check the SDK. The ctor variant for stdio-on-process is usually `new StdioServerTransport()` (uses real stdin/stdout automatically). If so, drop the `using hostStdin/hostStdout` pair and instantiate as `new StdioServerTransport()` or similar. The design intent: the transport owns the process's real stdin/stdout.
