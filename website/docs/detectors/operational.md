# Operational detectors

The 10 operational detectors flag UX / quality issues — repetition loops, truncated output, blank responses, placeholder text, wrong-language drift.

Highlights:

- **`OPS-01` BlankResponse** — empty or whitespace-only assistant message
- **`OPS-02` TruncatedOutput** — incomplete sentence at the end of a response
- **`OPS-04` RepetitionLoop** — same phrase repeated N+ times
- **`OPS-05` PlaceholderText** — `[TODO]`, `[insert-here]`, `lorem ipsum`
- **`OPS-08` WrongLanguage** — output language differs from input
